# User Stories for VibeBoard

## Project Management
- [x] I want to create a new project with a title and description.
- [x] I want to switch between different projects easily via the header navigation.
- [x] I want to see high-level statistics (counts) in the header at all times.
- [x] I want to delete a project when it is no longer needed.
- [x] I want to manage many projects from a single dashboard view.
- [x] I want to "park" a project with context notes for future use.
- [x] I want to see hidden/parked status and notes in the project list.
- [x] I want to resume a parked project and immediately see its notes.

## Task Management
- [x] I want to view all tasks within my current project in a table.
- [x] I want to create new tasks using a modal form.
- [x] I want to edit the details of an existing task quickly.
- [x] I want to delete tasks from the system.
- [x] I want to track task progress through a clear status lifecycle (Backlog -> In Progress -> QA-Review -> Done).
- [x] I want to see a visual 7-stage progression bar for active tasks.
- [x] I want to define dependencies between different tasks.
- [x] I want to record results from task executions in the system logs.
- [x] I want the system to automatically retry failed actions on specific tasks.

## Worker & Swarm Management
- [x] I want to see active workers and identify who is working on what currently.
- [x] I want to manually assign a worker to a specific task for clarity.
- [x] I want my dashboard to update the worker status automatically every 5 seconds.
- [x] I want a "Swarm" activity view to see overall team progress.

## Dashboard & UX
- [x] I want all count data (tasks, completion rates) scoped to my current project.
- [x] I want to see 5 key metrics displayed as cards on the main screen.
- [x] I want a doughnut chart to show the distribution of task statuses overhead.
- [x] I want the dashboard to auto-refresh every 30 seconds.
- [x] I want clear loading indicators while data is being fetched.
- [x] I want the site to have a consistent, high-quality dark theme.
